# SimpleWebServer

## Getting started
```shell
git clone https://github.com/Albertus82/SimpleWebServer.git
cd SimpleWebServer
mvn clean package
cd target
```

* **Linux**
  ```shell
  chmod 755 simple-web-server.sh
  ./simple-web-server.sh 8080 ~/webserver/
  ```

* **Windows**
  ```shell
  SimpleWebServer 8080 c:/webserver/
  ```
